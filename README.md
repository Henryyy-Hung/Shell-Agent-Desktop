# Shell Agent 桌面端

Shell Agent 是一个基于 Electron 和 React 的桌面应用程序，
旨在提供一个用户友好的界面来使用 AI Agent 控制远程终端，提供更快捷的操作体验。

## 先决条件

- NPM 11.6.1
- Node v22.20.0

## 安装

克隆仓库并安装依赖：

```bash
npm install
```

**调试指南 [debugging guide](https://github.com/electron-react-boilerplate/electron-react-boilerplate/issues/400)**

## 启动开发环境

在 `dev` 环境下启动应用：

```bash
npm start
```

## 打包生产版本

为本地平台打包应用：

```bash
npm run package
```
